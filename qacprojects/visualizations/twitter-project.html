Colophon

These Tweets are collected from the Twitter Streaming API using tweepy, stored in MySQL, manipulated using pandas, and visualized using D3 and NVD3.

Volume of Tweets

This line chart shows how the volume of Tweets sent by members of Congress changes over time. Use the grey buttons to change the aggregation period and the small chart to change the date range. Scroll over the line in the large chart for frequency data.

All timestamps are in Eastern Daylight Time.

Tweet Sentiment by Party

This line chart shows the change in Tweet sentiment over time by party. A lexicon is used to assign positive and negative values to the Tweet text. Sentiment is calculated as the positive value minus the absolute value of negative value.
